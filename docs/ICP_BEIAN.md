# ICP 备案完全指南

本指南详细说明如何为域名 `aidaquanji.com` 完成 ICP 备案，以便在中国大陆使用。

## 📋 什么是 ICP 备案？

ICP（Internet Content Provider）备案是中国大陆对网站的强制性管理要求。

**重要说明：**
- ✅ **完成备案后**，网站可以通过域名在国内正常访问
- ❌ **未备案的域名**，在国内无法通过域名访问（会被运营商拦截）
- ⏱️ **备案周期**：通常需要 7-20 个工作日

### 步骤 2：填写主体信息（15分钟）

#### 2.1 选择备案类型

**首次备案：**
- 如果您和您的单位从未做过备案，选择"首次备案"

**其他情况：**
- 新增网站：已有备案号，添加新网站
- 新增接入：网站已在其他服务商备案，转入阿里云
- 变更备案：修改已有备案信息

#### 2.2 填写主体信息

**个人备案填写：**
```
主办单位性质：个人
主办单位名称：您的真实姓名
主办单位证件类型：身份证
主办单位证件号码：您的身份证号
主办单位通信地址：您的真实居住地址
主办单位负责人姓名：您的真实姓名
主办单位负责人证件类型：身份证
主办单位负责人证件号码：您的身份证号
主办单位负责人手机号：您的手机号（实名认证）
主办单位负责人邮箱：您的邮箱
```

**企业备案填写：**
```
主办单位性质：企业
主办单位名称：营业执照上的公司名称
主办单位证件类型：工商营业执照
统一社会信用代码：营业执照上的代码
主办单位通信地址：营业执照上的地址
主办单位负责人姓名：法人姓名
主办单位负责人证件类型：身份证
主办单位负责人证件号码：法人身份证号
主办单位负责人手机号：法人手机号
主办单位负责人邮箱：法人邮箱
```

#### 2.3 上传证件

- 上传身份证正反面照片（或营业执照）
- 确保照片清晰、四角完整、无反光
- 格式：JPG/JPEG/PNG
- 大小：不超过 4MB

#### 3.1 基本信息

```
网站名称：AI工具大全

#### 3.2 网站负责人信息

**个人备案：**
- 网站负责人与主体负责人一致，勾选"同主办单位负责人"

**企业备案：**
```
网站负责人姓名：网站负责人（可以是法人或其他员工）
网站负责人证件类型：身份证
网站负责人证件号码：身份证号
网站负责人手机号：手机号

#### 3.3 上传网站负责人照片

- 白底免冠照片
- 穿着得体
- 清晰可见五官
- 格式：JPG/JPEG
- 大小：不超过 4MB

---

### 步骤 4：上传资料（10分钟）

#### 4.1 真实性核验单（已电子化）

阿里云已实现电子化，无需纸质核验单，但需要：

1. 人脸识别（使用阿里云 APP）
2. 签署承诺书

#### 4.2 其他材料（根据要求）

某些地区可能需要额外材料：
- 网站建设方案书
- 域名证书
- 授权书（企业备案，法人不是网站负责人时）

---

### 步骤 5：提交初审（1分钟）

1. 仔细检查所有信息
2. 勾选"我已阅读并同意《信息安全管理协议》"
3. 点击"提交备案"

#### 初审时间：
- 阿里云初审：1-2 个工作日
- 会收到短信和邮件通知

---

### 步骤 6：短信核验（即时）

#### 6.1 收到核验码

初审通过后，您会收到工信部发送的短信核验码（2条）：
- 主体负责人手机号
- 网站负责人手机号（企业备案）

短信格式：
```
【工信部】您的备案申请已提交...验证码：ABC123
请在24小时内登录 beian.miit.gov.cn 完成核验
```

#### 6.2 完成核验

1. 访问 [工信部备案管理系统](https://beian.miit.gov.cn/)
2. 点击"短信核验"
3. 输入：
   - 手机号码
   - 短信验证码
   - 证件号码后6位
4. 点击"提交"

⚠️ **必须在 24 小时内完成，否则备案会被退回！**

---

### 步骤 7：等待管局审核（7-20天）

#### 7.1 审核流程

```
提交 → 阿里云初审(1-2天) → 短信核验(24小时内) → 管局审核(7-20天) → 备案成功
```

#### 7.2 审核状态查询

- 登录 [阿里云备案系统](https://beian.aliyun.com/)
- 查看备案进度

#### 7.3 审核结果

**通过：**
- 收到短信和邮件通知
- 获得备案号：如 "京ICP备12345678号-1"

**被退回：**
- 根据退回原因修改
- 重新提交

## ⚠️ 重要注意事项

### 1. 网站名称规范

**个人备案禁止：**
- ❌ 不能涉及行业词（如：AI、工具、网站、平台等）
- ❌ 不能涉及地域（如：中国、北京等）
- ❌ 不能涉及经营性质（如：商城、交易等）

**个人备案建议：**
- ✅ 某某的个人网站
- ✅ 某某的技术分享
- ✅ 某某的学习笔记
- ✅ 某某的兴趣爱好

**企业备案：**
- ✅ 可以使用公司全称或简称
- ✅ 可以使用产品或服务名称

### 2. 备案期间注意

- 🚫 **网站不能访问**：备案期间，域名不能解析到服务器，否则会被拒绝
- ✅ **使用 IP 访问**：可以通过 IP 地址测试网站
- ✅ **准备正式内容**：管局会检查网站内容

### 3. 网站内容要求

- ✅ 不能有违法违规内容
- ✅ 不能有前置审批项目（如：新闻、视频、游戏等）
- ✅ 内容要与备案主体一致
- ✅ 页面要有版权信息和备案号

### 4. 首页底部必须添加备案号

备案成功后，必须在网站首页底部添加备案号和链接：

```html
<footer>
  <a href="https://beian.miit.gov.cn/" target="_blank">
    京ICP备12345678号-1
  </a>
</footer>
```

### 1. 备案信息变更

信息发生变化时必须及时变更：
- 主体信息变更（地址、电话等）

不再使用域名或网站时，应该注销备案：
- 主体注销：注销该主体下所有备案

### 3. 定期核查

工信部会定期核查备案信息：
- 确保网站能正常访问
- 确保备案信息准确
- 确保网站内容合规

## 🐛 常见问题

### Q1: 个人可以备案商业网站吗？

**A:** 不可以。个人备案只能用于非经营性网站。如果网站涉及经营（如：广告、交易、会员付费等），必须使用企业备案。

### Q2: 一个主体可以备案多少个网站？

**A:**
- 个人：不同省份限制不同，通常 2-5 个
- 企业：一般没有限制

### Q3: 备案期间网站能访问吗？

**A:** 不能。备案期间域名不能解析到服务器，但可以通过 IP 访问测试。

### Q4: 备案被退回怎么办？

**A:**
1. 查看退回原因
2. 根据要求修改信息或材料
3. 重新提交
4. 如有疑问，联系阿里云客服

### Q5: 备案号丢失了怎么办？

**A:**
1. 登录 [工信部备案系统](https://beian.miit.gov.cn/)
2. 点击"备案信息查询"
3. 输入域名或主体信息查询

### Q6: 备案成功后多久可以使用域名？

**A:** 立即可以使用。备案成功后：
1. 配置域名解析
2. 配置 Nginx
3. 在网站首页添加备案号
4. 网站即可正常访问
